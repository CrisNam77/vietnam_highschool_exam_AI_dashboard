# ExamData AI Frontend

Frontend chinh cua dashboard. Nen chay tu thu muc goc bang:

```cmd
run_frontend.cmd
```

Hoac chay thu cong trong thu muc nay:

```bash
npm install
npm run dev
```

Backend mac dinh:

```text
http://localhost:8001
```

Doi backend URL bang bien moi truong:

```powershell
$env:NEXT_PUBLIC_API_URL="http://localhost:8001"
npm run dev
```
